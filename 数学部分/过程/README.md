使用统一的纯文本符号约定，不包含任何 LaTeX 控制符：

| 符号类型 | 表示方法 | 示例 |
|----------|----------|------|
| 向量 | `vec(x)` | `vec(w)`, `vec(theta)` |
| 矩阵 | `mat(A)` | `mat(W)`, `mat(X)` |
| 分数 | `a/b` | `1/N`, `sigma^2/2` |
| 下标 | `x_i` | `w_1`, `x_D` |
| 上标 | `x^2` | `theta^*`, `w^ML` |
| 乘法 | `*` | `a * b` |
| 范数 | `\|vec(x)\|` | `\|vec(w)\|^2` |
