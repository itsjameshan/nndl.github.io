# 神经网络与深度学习 课程教学安排

## 课程信息
- **总周数**：16 周
- **每周课时**：3 节连上（每节 45 分钟，共 135 分钟）
- **总课时**：48 课时
- **教材**：
  - 理论：邱锡鹏《神经网络与深度学习》（简称"理论书"）
  - 实践：邱锡鹏《神经网络与深度学习——案例与实践》（简称"实践书"）

## 设计思路

理论书共 15 章 + 附录，实践书共 8 章。实践书的 8 章与理论书的对应关系如下：

| 实践书 | 理论书 |
|--------|--------|
| 第1章 实践基础 | — （框架与工具入门） |
| 第2章 机器学习概述 | 第2章 机器学习概述 |
| 第3章 线性分类 | 第3章 线性模型 |
| 第4章 前馈神经网络 | 第4章 前馈神经网络 |
| 第5章 卷积神经网络 | 第5章 卷积神经网络 |
| 第6章 循环神经网络 | 第6章 循环神经网络 |
| 第7章 网络优化与正则化 | 第7章 网络优化与正则化 |
| 第8章 注意力机制 | 第8章 注意力机制与外部记忆 |

课程采用"理论先行、实践跟进"的模式，每个主题先讲理论，再安排实践课。理论书中第1章（绪论）、第9–15章根据课时情况选讲。
